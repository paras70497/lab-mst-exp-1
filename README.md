# lab-mst-exp-1

React Counter Application using `useState`.

## Features
- Display counter value
- Increment button
- Decrement button

## Tech Stack
- React
- Vite

## Run Locally
1. Install dependencies:
   npm install
2. Start development server:
   npm run dev
3. Open:
   http://localhost:5173

## Folder Structure
- src/components/Counter.jsx
- src/App.jsx
- src/main.jsx
